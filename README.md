# Flask Expense Tracker

A web-based application built with Flask to help users track expenses, manage budgets, and visualize spending habits through interactive charts.

## 🚀 Features
- **Track Expenses:** Log title, amount, date, and category for every spend.
- **Data Visualization:** Interactive charts to see spending by category and trends over time.
- **Expense Management:** Add, view, and delete expense records easily.
- **Filters:** Search through your history using specific date ranges.

## 📂 Project Structure
- `app.py`: The main Python file containing Flask routes and logic.
- `templates/`: Directory containing the HTML files for the web interface.
- `requirements.txt`: A list of all Python libraries needed to run the app.

## 🛠️ Installation & Setup

### 1. Clone the repository
`git clone https://github.com/aadityamahajn`
`cd expense_tracker`

### 2. Create a Virtual Environment
`python -m venv venv`

**Activate on Windows:**
`venv\Scripts\activate`
**Activate on Mac/Linux:**
`source venv/bin/activate`

### 3. Install Dependencies
`pip install -r requirements.txt`

### 4. Run the Application
`python app.py`

The app will be available at: http://127.0.0.1:5000
